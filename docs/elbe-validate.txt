elbe-validate(1)
================

NAME
----
elbe-validate - validate xml file.

SYNOPSIS
--------
[verse]
'elbe validate' <xmlfile>


DESCRIPTION
-----------
This command validates the xmlfile provided, and prints out errors.


OPTIONS
-------
<xmlfile>::
	The xml file to validate.



EXAMPLES
--------

* Validate 'project.xml'
+
------------
$ elbe validate project.xml
------------
